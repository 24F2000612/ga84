# Supply Chain Correlation Analysis

**Email:** 24f2000612@ds.study.iitm.ac.in

This repository contains the required files for correlation heatmap analysis using Excel:

## Files

- **correlation.csv** – Exported correlation matrix from Excel  
- **heatmap.png** – Heatmap screenshot (Red–White–Green conditional formatting, 400–512 px)  
- **supply_chain.csv** – Input dataset (Supplier_Lead_Time, Inventory_Levels, Order_Frequency, Delivery_Performance, Cost_Per_Unit)  
- **generate_correlation.py** – Script to compute correlation + heatmap (optional)

## Steps Performed (Excel)
1. Imported dataset into Excel  
2. Enabled **Data Analysis ToolPak**  
3. Used **Correlation** tool to generate correlation matrix  
4. Exported matrix to **correlation.csv**  
5. Created heatmap using:  
   Home → Conditional Formatting → Color Scales → Red–White–Green  
6. Took screenshot (within 400–512 px) → **heatmap.png**
